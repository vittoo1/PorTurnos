Naturaleza Resistente

Naturaleza Resistente : Diseño y desarrollo de una tienda online especializada en cactus y suculentas. El sitio cuenta con una interfaz moderna de usuario, catálogo dinámico, filtro por especie, y carrito de compras funcional. Se priorizó una paleta de colores naturales con acento verde para transmitir calma y conexión con la naturaleza. El proyecto fue construido con HTML5, CSS3, Bootstrap y funcionalidades de e-commerce integradas con JavaScript.

Tsunagi Ramen

Restaurante Futurista Online: Landing page interactiva para un restaurante de ramen con temática sci-fi. El diseño UX/UI destaca por su estilo galáctico con degradados y tipografía bold. Incluye navegación animada, secciones para menú, recetas y localización. El sitio está optimizado para dispositivos móviles y desarrollado con tecnologías modernas como Figma para prototipado, React.js en frontend y animaciones CSS para una experiencia envolvente..

Ghifud

Plataforma de Delivery Estilo Anime: Aplicación web de entrega de comida con estética inspirada en Studio Ghibli. El proyecto combina ilustración personalizada con diseño funcional, incorporando secciones de registro, login, listado de servicios y botón de llamada a la acción. Se desarrolló con enfoque mobile-first, integrando Firebase para autenticación y React + Tailwind CSS para un desarrollo ágil y visualmente atractivo.
